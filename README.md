# Nine Flags

Browser-based clone of *Battle Line* (Knizia / GMT) for personal play. Plays on desktop and on a mobile phone in the browser.

Phase 1 is two humans on one device (pass-and-play). Online multiplayer and an AI opponent come in later phases.

See `docs/decisions-so-far.md` for design notes and `mockups/layout-preview.html` for a static layout preview.
